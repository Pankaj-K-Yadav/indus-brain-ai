# Datasets

Industrial documents used by INDUS-BRAIN AI.

- `raw/` — original, unprocessed source documents (PDFs, manuals, logs). Git-ignored; keep local only.
- `processed/` — cleaned / chunked / embedded-ready artifacts. Git-ignored.

> Large data files are intentionally excluded from version control via the root `.gitignore`. Only `.gitkeep` placeholders are committed so the folder structure is preserved.
